Standard Installation

For detailed instructions, see the full Installation Guide in our GitHub repository.

Quick Start

# Clone the repository
git clone https://github.com/replit/staples-brain.git
cd staples-brain

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env file with your configuration

# Initialize the database
python -m backend.database.scripts.run_scripts --setup

# Start the application
python run.py

Development Environment Setup

# Clone the repository
git clone https://github.com/replit/staples-brain.git
cd staples-brain

# Set up virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install development dependencies
pip install -e ".[dev]"

# Set up environment variables
cp .env.example .env
# Edit .env file with your configuration

# Initialize the database with sample data
python -m backend.database.scripts.run_scripts --setup --sample-data

# Start the development server
python run.py

Troubleshooting Common Issues

For detailed instructions, see the full Troubleshooting Guide.

If you're experiencing database connection issues:

  • Verify DATABASE_URL environment variable is correct
  • Check that PostgreSQL is running
  • Ensure the database exists and is accessible
  • Check network connectivity to the database server

Common OpenAI API issues:

  • Invalid API key or expired key
  • Rate limit exceeded
  • Insufficient permissions for chosen models

If you're having issues with package installation:

  • Try upgrading pip: pip install --upgrade pip
  • Install packages incrementally using the setup script: python setup_local_env.py
  • Check for package version conflicts
